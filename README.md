## 프로젝트의 실행 방법

### 패키지 설치

```
# NPM
npm install
# Yarn
yarn install
```

### 어플리케이션 실행

다음의 명령어 실행 후 <http://localhost:3000> 로 접속

```
# NPM
npm start
# Yarn
yarn start
```
